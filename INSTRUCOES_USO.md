## 💡 Recomendação

**Para aproveitar melhor AGORA:**

1. Abra o `index.html` em seu navegador
2. Navegue até a Semana 01
3. Veja o exemplo completo de rubricas e soluções
4. Use como base para entender o que esperar dos exercícios

**Para criar guias adicionais:**

1. Leia o `METODOLOGIA_GUIAS.md`
2. Escolha uma semana para trabalhar
3. Abra o arquivo original (ex: semana06.html)
4. Use o template e adapte para os 3 exercícios daquela semana
5. Copie a estrutura HTML do semana01_guia.html
