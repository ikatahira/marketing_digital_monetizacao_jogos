# 📚 Guias de Estudo - Marketing Digital e Monetização para Jogos

## 📋 Sobre este Material

Este conjunto de guias de estudo foi criado para complementar o curso de Marketing Digital e Monetização para Jogos da FATEC Ourinhos. Cada guia contém:

### ✅ Rubricas Detalhadas de Avaliação
- Critérios claros de pontuação (Insuficiente, Básico, Proficiente, Excelente)
- Tabelas organizadas para cada exercício
- Pontuação de 0-10 para cada critério

### 💡 Soluções Completas
- Exemplos detalhados de como realizar cada exercício
- Cases reais da indústria de jogos
- Dados e métricas atualizadas
- Templates e estruturas prontas para uso

### 📊 Exemplos Práticos
- Referências a jogos reais (AAA, indie, mobile)
- Campanhas de marketing documentadas
- Análises de mercado com dados concretos
- Estratégias comprovadas do mercado

## 🗂️ Estrutura dos Guias

Cada guia de semana contém:

1. **Cabeçalho** - Título da semana e tema principal
2. **Exercício 1** - Com rubrica e solução detalhada
3. **Exercício 2** - Com rubrica e solução detalhada
4. **Exercício 3** - Com rubrica e solução detalhada
5. **Navegação** - Links para semana anterior/próxima e índice

## 📖 Como Usar

### Para Estudantes:
1. Leia o guia ANTES de fazer os exercícios
2. Use as rubricas para entender o que é esperado
3. Consulte as soluções apenas DEPOIS de tentar
4. Adapte os exemplos ao seu contexto

### Para Professores:
1. Use as rubricas como base para avaliação
2. Personalize os critérios conforme necessário
3. Os exemplos podem ser usados como gabarito
4. Atualize dados conforme o mercado evolui

## 📁 Arquivos Disponíveis

- `index.html` - Página principal com todas as semanas
- `semana01_guia.html` - Introdução ao Marketing Digital
- `semana02_guia.html` - Mercado Brasileiro de Jogos
- ... (continua para todas as 19 semanas)

## ⚠️ Nota Importante

Os dados e exemplos são baseados em informações públicas até início de 2025. Sempre verifique dados atualizados para análises de mercado.
